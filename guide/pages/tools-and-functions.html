Tools and Functions

The web app UI has many different methods of displaying tools and functions. But choosing the right one for the right type of tool is important.

The global toolbar

Placement of global tools

The global toolbar is for all tools that are relavant no matter where you are on the site. These are usually the same or atleast similar for every app. such as:

  • User profile
  • Messages
  • Alerts
  • Theme switcher

The theme switcher is the only compulsory function in the global toolbar. This allows users to toggle between the dark theme and light theme depending on their lighting environment.

Contextual floating action menu

The floating action menu

The floating action button expands on click to reveal more options (max 3). However it can be just used as one single button. This is for any options that relate directly to that section. Such as adding more content like a new driver or vehicle.

The floating button is not compulsory. Do not use it if there is not a good reason to do so, and only if it is 100% obvious what it will do.

Contextual toolbar

Page specific contextual tools

The contextual toolbar is placed beneath the tabs and (if tabs are used) placed in the ajax page content itself. It is specifically for tools that are specific for the page content beneath them such as search fields, filters and modifiers.

Be careful not to overuse this toolbar. You can only fit 2 items at the most due to smaller screen devices. They should always be left or right aligned. Not centre aligned.
